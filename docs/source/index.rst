Welcome to the documentation for STADiffuser - a versatile simulator transforming spatial transcriptomic data analysis.

By accurately simulating spatial transcriptomic data, STADiffuser empowers users with the means to explore and analyze gene expression within precise spatial contexts. This documentation will guide you through the features and functionalities of STADiffuser, showcasing its ability to model multiple samples and 3D coordinates.
